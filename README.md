# Site João Emanuel

Estrutura do projeto:

```
index.html          → página inicial
quem-sou.html        → página "sobre mim"
moove.html           → página "Conheça a Moove"
assets/
  css/style.css      → estilos do site
  js/script.js       → animações e interações
  img/               → fotos, logo da Moove e ícones do site
```

Basta abrir `index.html` no navegador ou publicar a pasta inteira
(ex.: GitHub Pages, Vercel, Netlify) — todos os caminhos já estão
relativos à raiz do projeto.
